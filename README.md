# 🎓 Student Performance Prediction using Decision Tree

## 📌 Project Overview

This project analyzes student performance data and builds a **Decision Tree Classification** model to predict whether a student is likely to **Pass** or **Fail** based on demographic and educational features.

The project also includes data preprocessing, exploratory data analysis (EDA), visualization, outlier detection, feature encoding, and model evaluation.

---

## 📂 Dataset

The project uses the **Student Performance Dataset**.

The dataset contains information such as:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

---

## 📊 Data Preprocessing

The following preprocessing steps were performed:

- Removed unnecessary columns
- Checked missing values
- Removed duplicated records
- Removed numerical outliers using the IQR method
- Encoded categorical variables using LabelEncoder
